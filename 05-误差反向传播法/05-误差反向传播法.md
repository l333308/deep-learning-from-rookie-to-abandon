# 链式法则 chain rule
先看复合函数，由多个函数复合而成的函数。
z = (x+y)^2,可拆分为以下两个函数
1) z = t^2
2) t = x+y
在求导时，该链式法则也试用。z对x的导数，就是z对t的导数乘以t对x的导数。
1）的导数是2t。推导：( (t+h)^2 - t^2 ) / h = 2t + h。h趋近于0。所以，z对t的导数是2t。
2）的导数是1
所以，z对x的导数是2t * 1 = 2(x+y)
